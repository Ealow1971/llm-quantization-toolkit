# LLM Quantization Toolkit 📉

![PyTorch](https://img.shields.io/badge/pytorch-2.0+-EE4C2C.svg) ![LLM](https://img.shields.io/badge/llm-optimization-blueviolet.svg)

Comprehensive tools for compressing Large Language Models with minimal accuracy loss.

## Supported Methods
- **GPTQ**: Post-training quantization.
- **AWQ**: Activation-aware Weight Quantization.
